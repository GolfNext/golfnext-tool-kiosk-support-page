# GolfNext Brand Plugin

A Claude plugin that gives the AI complete knowledge of GolfNext's brand identity.

## What it does

Once installed, Claude automatically follows GolfNext brand guidelines whenever creating content — emails, LinkedIn posts, presentations, product copy, customer communication, and more.

## Included

- **Tone of Voice** — Credible, knowledgeable, human
- **Core Narrative** — The "Open Future of Golf" positioning
- **Design Guidelines** — Colors, typography, logo rules, photography
- **Logo Assets** — All official logos in SVG and PNG (6 variants)

## Installation

### For individual users
1. Download the plugin
2. Open Claude Desktop > Settings > Capabilities
3. Install the plugin

### For admins (org-wide)
1. Go to claude.ai > Admin Settings > Capabilities
2. Upload the plugin to provision it for all users

## Usage

Just ask Claude to create GolfNext content — it will automatically use the correct tone, colors, messaging, and logos.
